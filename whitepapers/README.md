# White Papers

This directory contains the full collection of xlab white papers. Each subfolder holds a `paper.md` (the full paper) and a `README.md` (executive summary).

| Paper | Domain | One-liner |
|---|---|---|
| [**OIRP**](oirp/) | Web Infrastructure | An open protocol that decouples naming from DNS for AI-agent-driven navigation. |
| [**HCE**](hce/) | AI Economics | A six-layer framework mapping where value forms when AI commoditises general reasoning. |
| [**B.U.D**](bud/) | Digital Public Infrastructure | A policy proposal to extend India's DPA (Aadhaar, UPI) into cloud deployment for every builder. |
